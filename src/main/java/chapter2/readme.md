# 一切都是对象
## 1、主要类型

| 类型 | 长度 | 默认值|
| :--- | :--- | :--- |
|boolean|1| false|
|char|16|'\u000'(null)|
|byte|8|(byte)0|
|short|16|(short)0|
|int|32|0|
|long|64|0L|
|float|32|0.0f|
|double|64|0.0d|
**所有的数值类型都是有符号的，没有无符类型**
局部变量(方法中)没有初始值，而类的成员变量是有初始值的[FieldInit.java](./FieldInit.java)
